# mgDIF
Supplementary material for the paper "Differential item functioning detection across multiple groups"

Folder Applications:
- AnalysisTIMSSG4.r: code to perform the analyses
- DataApplication.RData: includes the TIMSS data

Folder Functions:
- functions_IndLik.r: R code to estimate the parameters
- IndLik.cpp: c++ code to estimate the parameters

Folder Simulations:
- eqLIK_b1E.RData: file including estimates obtained in the second application
- FzSim.r: auxiliary functions
- simulations1_position.r: R code to perform the first simulation study
- simulations2_groups.r: R code to perform the second simulation study
